# <a name="activating-windows-10-subscriptions"></a>Активация подписок на Windows 10

Microsoft 365 бизнес премиум, Office 365 для образования, Microsoft 365 и Enterprise предоставляют цифровые лицензии для Windows 10 в рамках подписки. Чтобы активировать подписку, на вашем устройстве уже должна быть активирована Windows 10 Pro (1703 или более поздней версии). Полный список требований к активации подписки Windows 10 см. в Windows 10 Требования к активации [подписки.](https://docs.microsoft.com/windows/deployment/windows-10-subscription-activation#requirements)
