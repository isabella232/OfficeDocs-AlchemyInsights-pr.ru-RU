# <a name="add-update-or-delete-payment-method-in-azure"></a>Добавление, обновление или удаление метода оплаты в Azure

Администратор учетной записи может добавить новую кредитную карту, обновить существующую кредитную карту или удалить не используемую кредитную карту. Выполните следующие действия, чтобы добавить или заменить метод оплаты:

1. Во входе на [портал Azure](https://portal.azure.com/) в качестве [администратора учетной записи.](https://docs.microsoft.com/azure/billing/billing-subscription-transfer?WT.mc_id=Portal-Microsoft_Azure_Support#whoisaa) Управление **затратами на поиск + выставление счета**
2. Выберите подписку, в которая вы хотите добавить кредитную или дебетовую карту.
3. Выберите **Методы оплаты**
4. В левом верхнем углу нажмите "+", чтобы добавить карту. В правой части экрана появится форма кредитной карты. Ввод сведений о кредитной или дебетовой карте
5. Чтобы сделать эту карту активным методом оплаты, проверьте поле рядом с Сделать это мой активный метод оплаты выше формы. Эта карта станет активным инструментом для оплаты всех подписок, использующих ту же карту, что и в выбранной подписке. Нажмите **Далее**.
6. Чтобы **использовать другую кредитную карту,** проверьте поле рядом с картой, в который вы хотите внести активный способ оплаты.
7. Нажмите **кнопку Установите активную**
