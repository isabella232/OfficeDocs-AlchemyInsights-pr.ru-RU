# <a name="single-sign-on-sso-in-microsoft-edge"></a>Однорегистрный вход (SSO) в Microsoft Edge

Microsoft Edge поддерживает следующие типы одного входного знака (SSO):
- SSO на основе PRT
- Бесшовный SSO
- SSO на основе WIA
