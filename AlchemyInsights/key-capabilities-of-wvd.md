# <a name="key-capabilities-of-windows-virtual-desktop"></a>Основные возможности Виртуального рабочего стола Windows

Виртуальный рабочий стол Windows — это служба виртуализации рабочих столов и приложений, работающая в облаке. Виртуальный рабочий стол Windows позволяет:

- Настройка развертывания нескольких сеансов Windows 10, которое обеспечивает полное Windows 10 масштабируемости
- Виртуализация Приложения Microsoft 365 и оптимизация для работы в много пользовательских виртуальных сценариях
- Предоставление Windows 7 виртуальных настольных компьютеров с бесплатными расширенными обновлениями безопасности (ESU)
- Принесите существующие службы удаленного рабочего стола (RDS) и Windows серверов и приложений на любой компьютер
- Управление Windows 10, Windows Server и Windows 7 настольных компьютеров и приложений с единой опытом управления
- Создание полной среды виртуализации рабочего стола в подписке Azure без необходимости запуска дополнительных серверов шлюза
- Публикуйте столько пулов хостов, сколько необходимо для размещения разнообразных рабочих нагрузок
- В галерее Azure принесите собственные изображения для рабочих нагрузок и тестов
- Сокращение затрат с помощью ресурсов с несколькими сеансами
- Предоставление индивидуального владения с помощью персональных (постоянных) настольных компьютеров

Дополнительные сведения см. в разделе [Основные сведения о Виртуальном рабочем столе Windows](https://go.microsoft.com/fwlink/?linkid=2127033) и [Основные возможности](https://go.microsoft.com/fwlink/?linkid=2127033).
