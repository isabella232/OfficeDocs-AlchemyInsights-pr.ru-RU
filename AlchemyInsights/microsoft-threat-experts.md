# <a name="microsoft-threat-experts"></a>Microsoft Threat Experts

Microsoft Threat Experts — это служба управляемой охоты на угрозы в Microsoft Defender для конечной точки.  Мы сотрудничаем с Вашим Центром операций безопасности для мониторинга, анализа и поддержки для выявления критических угроз в уникальной среде и реагирования на них.

Эта служба управляемой охоты на угрозы предоставляет экспертные сведения и данные с помощью этих двух возможностей: целевого уведомления об атаке и доступа к экспертам по запросу. Сведения о Microsoft Defender для конечной точки см. в [материале Microsoft Threat Experts.]( https://docs.microsoft.com/microsoft-365/security/defender-endpoint/microsoft-threat-experts)

Сведения об управляемой службе охоты на угрозы в контексте защиты конечных точек организации, электронной почты и удостоверений см. в обзоре [Microsoft Threat Experts in Microsoft 365.](https://docs.microsoft.com/microsoft-365/security/mtp/microsoft-threat-experts?view=o365-worldwide)
