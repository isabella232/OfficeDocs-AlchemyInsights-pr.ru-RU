# <a name="help-prevent-winmaildat-attachments-in-email-messages-from-your-organization"></a>Справка о предотвращении вложений Winmail.dat в сообщениях электронной почты из организации

В качестве администратора попробуйте следующие действия:

1. Откройте [Центр администрирования Exchange](https://outlook.office365.com/ecp/)

2. Перейдите **к удаленным**  >  **доменам потока почты.**

3. Выберите удаленный домен по умолчанию с именем **Default** и нажмите кнопку **Изменить**.

4. В разделе **Использование формата с** богатым текстом выберите **Never**.

Дополнительные сведения см. в [тексте Укажите формат сообщений для удаленных доменов.](https://docs.microsoft.com/Exchange/mail-flow-best-practices/remote-domains/remote-domains#specifying-message-format)
