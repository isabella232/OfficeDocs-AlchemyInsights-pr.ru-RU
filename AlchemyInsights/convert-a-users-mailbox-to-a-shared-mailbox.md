# <a name="convert-a-user-mailbox-to-a-shared-mailbox"></a>Преобразование почтового ящика пользователя в общий почтовый ящик

При преобразовании почтового ящика отдельного пользователя в общий почтовый ящик все существующие элементы электронной почты и календаря сохраняются в общем почтовом ящике, к нему могут получить доступ несколько пользователей. Вы также можете преобразовать общий почтовый ящик обратно в пользовательский (частный) почтовый ящик.
  
Дополнительные сведения см. в раздел [Преобразование почтового ящика пользователя в общий почтовый ящик.](https://docs.microsoft.com/microsoft-365/admin/email/convert-user-mailbox-to-shared-mailbox)
