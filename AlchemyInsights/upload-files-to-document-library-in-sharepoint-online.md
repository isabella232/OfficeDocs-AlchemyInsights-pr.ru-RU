# <a name="upload-files-to-onedrive-or-sharepoint"></a>Upload файлы для OneDrive или SharePoint

Вы можете загружать файлы в библиотеку документов в SharePoint Online, просто перетаскив их с компьютера и сбросив их в библиотеку документов. С OneDrive вы можете синхронизировать файлы между компьютером и облаком, чтобы можно было добраться до файлов из любой точки. Вы также можете работать с синхронизированными файлами непосредственно в File Explorer и получать доступ к файлам даже в автономном режиме. Дополнительные сведения о загрузке и синхронизации см. в ссылке ниже.
