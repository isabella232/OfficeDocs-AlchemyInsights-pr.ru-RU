# <a name="activate-a-new-subscription-with-a-product-key"></a>Активация новой подписки с помощью ключа продукта

1. Чтобы активировать новую подписку с помощью ключа продукта, перейдите к [https://office.com/setupkey](https://office.com/setupkey) .

2. Введите ключ продукта и следуйте подсказкам для активации подписки. Если у вас возникают проблемы, просмотрите [список распространенных ошибок, связанных с ключом продукта, и способов их устранения](https://docs.microsoft.com/microsoft-365/commerce/product-key-errors-and-solutions).
