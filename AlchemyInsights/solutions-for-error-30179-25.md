# <a name="solutions-for-error-30179-25"></a>Решения для ошибки 30179-25

Вы можете увидеть ошибку в сообщении "Не удалось поток Office" (код ошибки [30179-25),](https://support.office.com/article/e40d3c7d-98f6-4284-94a0-882beaa44593?wt.mc_id=Alchemy_ClientDIA)если на жестком диске не будет места.
  
[Освободите место на жестком диске и](https://support.microsoft.com/help/12425/windows-10-free-up-drive-space) [переустановите Office.](https://portal.office.com/OLS/MySoftware.aspx)
