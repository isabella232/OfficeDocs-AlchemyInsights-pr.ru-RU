# <a name="limit-sharing-in-microsoft-365"></a>Ограничение общего доступа в Microsoft 365

Хотя вы не можете полностью отключить внутренний общий доступ или удалить кнопку Share с сайтов, существует множество способов ограничить доступ к Microsoft 365 для удовлетворения потребностей организации. Дополнительные сведения см. в [разделе Limit sharing in Microsoft 365.](https://docs.microsoft.com/Office365/Enterprise/microsoft-365-limit-sharing)
