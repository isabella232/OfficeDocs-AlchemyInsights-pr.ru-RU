# <a name="using-custom-policy-in-intune"></a>Использование настраиваемой политики в Intune

Intune имеет множество встроенных параметров для android, iOS, macOS и Windows устройств. Вы также можете создать настраиваемый профиль, а затем развернуть эти настраиваемые параметры на устройствах с помощью Intune. Настраиваемые профили предназначены для добавления параметров и функций устройств, которые не встроены в Intune.

[Настраиваемые параметры для android-устройств в Intune](https://docs.microsoft.com/intune/custom-settings-android)

[Настраиваемые параметры для устройств Enterprise Android в Intune](https://docs.microsoft.com/intune/custom-settings-android-for-work)

[Настраиваемые параметры для устройств iOS в Intune](https://docs.microsoft.com/intune/custom-settings-ios)

[Настраиваемые параметры для устройств macOS в Intune](https://docs.microsoft.com/intune/custom-settings-macos)

[Настраиваемые параметры для Windows 10 устройств в Intune](https://docs.microsoft.com/intune/custom-settings-windows-10)

[Настраиваемые параметры Windows Holographic for Business устройств в Intune](https://docs.microsoft.com/intune/custom-settings-windows-holographic)
