# <a name="microsoft-365-apps-crash-stop-responding-or-dont-launch"></a>Microsoft 365 сбоя приложений, остановки ответа или не запуска

Чтобы исправить Microsoft 365 приложения зависают или сбои, попробуйте следующие варианты:

- [Изучите возможные проблемы с надстройки,](https://support.office.com/article/powerpoint-isn-t-responding-hangs-or-freezes-652ede6e-e3d2-449a-a07f-8c800dfb948d#bkmk_addins) установленными для Office
- Проверьте, [конфликтуется](https://support.office.com/article/powerpoint-isn-t-responding-hangs-or-freezes-652ede6e-e3d2-449a-a07f-8c800dfb948d?ocmsassetID#bkmk_conflict) ли антивирусное программное обеспечение с Microsoft 365 приложениями
- Проверка или [установка последних обновлений для Office](https://support.office.com/article/update-office-and-your-computer-with-microsoft-update-2ab296f3-7f03-43a2-8e50-46de917611c5)
- [Ремонт Office](https://support.office.com/article/repair-an-office-application-7821d4b6-7c1d-4205-aa0e-a6b40c5bb88b) с помощью панели управления
- [Удалить и](https://support.office.com/article/uninstall-office-from-a-pc-9dd49b83-264a-477a-8fcc-2fdf5dbf61d8) переустановить Office
