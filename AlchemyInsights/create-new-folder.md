# <a name="create-a-new-folder-in-outlookcom"></a>Создание новой папки в Outlook.com

1. В **папках** в области навигации выберите **новую папку**.

2. Введите имя папки и нажмите кнопку Ввод.

Дополнительные дополнительные возможности см. [в сайте Create a folder in Outlook.com.](https://support.office.com/article/5fa8de74-3562-4729-ac1d-5599f470b25a?wt.mc_id=Office_Outlook_com_Alchemy)
